This is a flutter mobile application interacting with the python API fitness_nutrition_tracking_flask. As attahced in the image below, users can get personalized meal and exercise plans tailored according to their answer to the questionaire during sign in of the application as detailed more in the back end implementation. The mobile application has different dashboard elements as shown in the images below. After logging in the user starts at the home page of the application, where they can see options for meal, excercise, and water intake. From the dashboard users can also access the database of workouts and meals where they can search using different filtering parameters. When selecting meal plans, the users meal plan for the month is shown below and includes different meals for the day of breakfast, lunch, dinner, and snack provided in such a way to meet with the users fitness goals, including weight loss, maintenance, and gain. The user can also read blogs written by trainers and other users as part of the application. The images shown below summarize all the above features for one logged in user.
